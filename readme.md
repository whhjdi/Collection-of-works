## 涂鸦画板
[源码地址](https://github.com/whhjdi/canvas-demo)
[预览链接](https://whhjdi.github.io/canvas-demo/)

### 项目介绍
本项目主要使用canvas和原生JS实现.使用HTML5的input新增类型color背景色和画笔颜色的选取.使用 canvas和JS实现笔刷选择,橡皮擦,保存和清空等功能可隐藏的按钮选项,让界面更简洁,跑马灯式的动画效果,让界面更加灵动.

## 豆瓣搜索
[源码地址](https://github.com/whhjdi/canvas-demo)
[预览链接](https://whhjdi.github.io/douban-demo/)

### 项目介绍
  本项目主要使用jQuery和bootstrap实现.使用bootstrap的组件(导航栏,欢迎页,翻页)进行页面设计,兼容PC和移动端.使用豆瓣API进行搜索,并展示搜索结果.主要功能:搜电影,搜音乐,搜图书,也可以回车进行百度搜索.搜索结果使用flex布局,实现响应式的排列.

## 在线简历生成器(vue)
[源码地址](https://github.com/whhjdi/vue-resumeEdit)
[预览链接](https://whhjdi.github.io/vue-resumeEdit/dist/#/)

### 项目介绍
  本项目是基于vue开发的在线简历编辑器,初步实现简历编辑,登录,注册等功能
  使用vue-cli脚手架生成开发环境,使用vue,vue-router,ES6,Scss构建单页面应用
  使用Element-ui的部分组件和功能美化页面
  使用阿里使用图标库(icon-font)
  使用vue-particles展现背景动态效果
  使用LeanCloud作为后台数据库,实现登陆注册以及用户信息的保存
  采用Element-ui进行前端表单验证
